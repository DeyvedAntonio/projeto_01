O que é HTML

HTML(Hyper Text Markup Language) é uma linguagem de marcação de páginas web, ou seja, é uma linguagem que avisa ao navegador como o conteúdo deve ser exibido ao usuário.

O que são elemntos no HTML

Os elementos que definem a forma de visualização de algum conteúdo nas páginas de internet.

Elementos possuem tags de abertura e fechamente. Algumas tags não possuem tags de fechamento.

Títulos - h1 até h6

Os elementos títulos informam o grau de relevância de acordo com os títulos.

As tags:

  • h1, grau maior de importância
  • h2,
  • h3,
  • h4,
  • h5
  • e h6, menor grau de importância.

Link - a

O elemento cria uma forma de armazenar uma url e ao clicar nessa tags 'a' o usuário é redirecionado para essa página armazenada.

Linha - hr

A tag 'hr' insere na página uma linha de possuem características específicas para cada navegador.

Parágrafo - p

O elemento cria uma marcação de parágrafo no conteúdo do texto.

Lista não ordenada - ul

A tag: 'ul' cria uma lista não ordenada de conteúdos.

Itens de uma lista - li

Elemento cria os itens da lista. Essa tag é definida dentro da tag de lista.
